# Sprint 1 — MVP Auth + Matchs

## Objectif
Un fan peut **créer un compte**, se connecter et **voir la liste des matchs**.

## Stories (cibles)
- A1 Inscription (5)
- A2 Login/Logout (3)
- M1 Liste des matchs (5)

## Tâches techniques (exemples)
- API auth : register/login + token
- UI pages : `/register`, `/login`, `/account`
- API matchs : `GET /matches`
- Seed data : 10 matchs + 4 stades

## Critères de succès (Review)
- Démo en 3 minutes : inscription → login → liste matchs → logout.
