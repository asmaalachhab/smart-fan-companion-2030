# Sprint 2 — Billetterie + Sélection de places (2 semaines)

## Sprint Goal
Permettre de **réserver un billet** sur un match avec une **sélection de section/place**.

## Stories sélectionnées
- SFC-020: Voir les sections/prix d’un stade (3)
- SFC-021: Choisir une section + quantité (3)
- SFC-022: Création de ticket / réservation (API + DB) (5)
- SFC-023: Génération QR / code de réservation (3)
- SFC-024: Page “Mes Billets” (3)

## Notes

## Démo attendue
