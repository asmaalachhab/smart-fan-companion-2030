# Cérémonies Scrum (cadence recommandée)

Sprint recommandé : **2 semaines**.

## Sprint Planning (2h)
**Objectifs**
- Définir le **Sprint Goal** (objectif clair)
- Choisir les stories du backlog (selon capacité)
- Découper en tâches techniques

**Sorties**
- Sprint Backlog (stories + tâches)
- Sprint Goal

## Daily Scrum (15 min)
Chaque dev répond :
1) Qu’est-ce que j’ai fini depuis hier ?
2) Qu’est-ce que je fais aujourd’hui ?
3) Quels blocages ?

## Backlog Refinement (1h / semaine)
- Clarifier les stories
- Ajouter critères d’acceptation
- Estimer (story points)
- Découper les grosses stories

## Sprint Review (1h)
- Démo de ce qui marche **dans l’app**
- Feedback PO / utilisateurs
- Mise à jour du backlog

## Sprint Retrospective (45 min)
Format simple :
- **Start** : quoi commencer
- **Stop** : quoi arrêter
- **Continue** : quoi garder
