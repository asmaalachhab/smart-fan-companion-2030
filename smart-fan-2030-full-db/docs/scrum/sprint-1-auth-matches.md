# Sprint 1 — Auth + Matchs (2 semaines)

## Sprint Goal
Un fan peut **créer un compte**, se connecter, et voir la liste des matchs (avec détails).

## Stories sélectionnées
- SFC-010: Inscription utilisateur (email + mot de passe) (3)
- SFC-011: Connexion + session/token (3)
- SFC-012: Profil utilisateur (lecture) (2)
- SFC-020: Liste des matchs (pagination/filtre simple) (3)
- SFC-021: Détails match (stade, ville, date, équipes) (2)

## Tâches techniques (exemples)
- Backend: routes auth, validation, erreurs
- DB: table users + indexes
- Frontend: pages Login/Register + guard routes
- Tests rapides: auth + fetch matchs

## Démo attendue
- Register → Login → page “Matchs”
- Détails d’un match ouvrable
